O que é o grid layout

O grid layout é bidmencional, permitindo que o desenvolvedor possa separar o layout da pagina em linha e colunas, difenretemente do flexbox que é unidimencional ordenando os elementos em linha.

Felxgrid ou grid layout

O flexbox e o grid layout podem ser usados em conjunto. Voce pode aplicar o flexbox e elementos dentro de um layout de grade como o grid layout, aproveitando o melhor dos dois mundos.

A escolha entre flexbox e grid layout nao se trata de um substituir o outro, mas sim de entender as necessidades especificas do seu layout e escolher a tecnica que melhor atenda as suas necessidades.
