O bootstrap utiliza um sistema de cores contextuais para facilitar a marcação e a estilização de elementos com base em seu significado ou proposito. Esses seletores de classe de cores são usados para dar significado visual aos elementos html. Geralmente utilizamos essas ores em botões, alerta, barras de navegação e outros elementos html

As cores contextuais mais utilizadas são:

Este texto é importante.

Este texto representa o sucesso de alguma operação ou informação.

Este texto aparece em cor mais fechada, escura, derivada do cinza.

Este texto representa perigo. Geralmente na tonalidade vermelha.

Este texto representa atenção.

Este texto repesenta apenas informações.
