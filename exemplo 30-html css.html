Exemplo 30 - flexbox

O flexbox é um modelo de layout em css que foi projetado para facilitar a criação de layout complexos. Rle é especialmente util para criação de layouts responsivos.

Principais propriedades do FlexBox

Flex-Direction

a propriedade flex-direction permite alterar a direção na qual os elementos html serão exibidos. A seguir demonstramos o seu uso.

Exemplo prático

Flex-Direction: row

Item 1
Item 2
Item 3

Flex-Direction: row-reverse

Item 1
Item 2
Item 3

Flex-Direction: colunm

Item 1
Item 2
Item 3

Flex-Direction: colunm-reverse

Item 1
